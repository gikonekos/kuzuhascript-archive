# Kuzuha Script Perl CGI Archive

Historical preservation repository for the **Kuzuha Script** Perl CGI system.

This repository preserves the original distribution archives and related materials as historical documents. The purpose of this project is preservation, not modernization or redevelopment.

The original source code is kept in its original form whenever possible.

**Initial archival release:** 2026-07-01 20:00 JST (11:00 UTC)

## Historical Note

The original website intentionally presents a "403 Forbidden"-like top page. The actual contents are accessible via links from that page.

This style may look unusual today, but it was not uncommon on Japanese personal websites and CGI distribution sites of the late 1990s and early 2000s.

In other words:

> If you see a 403 page, try clicking the links anyway.

## 歴史的補足

配布元サイトのトップページは、意図的に「403 Forbidden」風の画面になっています。

しかし、ページ内のリンクを辿ると実際のコンテンツへ到達できます。

現在では少々トラップのように見えますが、これは1990年代後半から2000年代前半の日本の個人サイトやCGI配布サイトでは、さほど珍しくない作法（あるいは遊び心）でした。

つまり――

> **「403だからといって、諦めてはいけない。」**
