# Kuzuha Script Perl CGI Archive

Historical preservation repository for the **Kuzuha Script** Perl CGI system.

This repository preserves the original distribution archives and related materials as historical documents. The purpose of this project is preservation, not modernization or redevelopment.

The original source code is kept in its original form whenever possible.

**Initial archival release:** 2026-07-01 20:00 JST (11:00 UTC)

## Preserved Releases

* Kuzuha Script β24 (auto-link version) — 1999-08-06
* Kuzuha Script Final Beta #2 Patch Level 0.3 — 1999-11-19
* Kuzuha Script Rev.0.1 Preview 9 — 2000-09-03
* Kuzuha Script Rev.0.1 Preview 9 (update patch)
* Kuzuha Script Final Beta #2 pl0.3 + "Yousei" variant, dated 2000-03-06
* Kuzuha Script Rev.0.1 Preview 9
* Kuzuha Script + @Kuzuha Moe ver1.1.1

Note: two archives (`dauso0001.lzh` and `kscrr1p9.lzh`) both identify
themselves internally as "Rev.0.1 Preview 9," but they are distinct
deployed instances with different embedded server configuration —
not duplicates. Both are preserved as-is.

## How These Files Were Found

The releases listed above (β24, Final Beta #2 pl0.3, Rev.0.1 Preview 9,
Rev.0.1 Preview 9 update patch, and the "Yousei" 2000-03-06 variant)
were recovered from my CD-R backup media during an unrelated
hardware excavation project, and added to this archive on 2026-07-05.
They predate the previously known earliest preserved release and were
not sourced from the original download site listed below.

## Original Download Source

Archived snapshot:

* https://web.archive.org/web/20201213102054/https://qptn.x.fc2.com/up/index.html

Please note that some files in the Internet Archive snapshot appear to be incomplete or unavailable.

## Historical Note

The original website intentionally presents a "403 Forbidden"-like top page. The actual contents are accessible via links from that page.

This style may look unusual today, but it was not uncommon on Japanese personal websites and CGI distribution sites of the late 1990s and early 2000s.

In other words:

> If you see a 403 page, try clicking the links anyway.

## Preservation Policy

* Respect original copyright notices and author information.
* Do not modify the original source code.
* UTF-8 conversions, JavaScript ports, translations, and modern implementations should be developed in separate repositories.

## 歴史的補足

配布元サイトのトップページは、意図的に「403 Forbidden」風の画面になっています。

しかし、ページ内のリンクを辿ると実際のコンテンツへ到達できます。

現在では少々トラップのように見えますが、これは1990年代後半から2000年代前半の日本の個人サイトやCGI配布サイトでは、さほど珍しくない作法（あるいは遊び心）でした。

つまり――

> **「403だからといって、諦めてはいけない。」**

## 保存方針

* 原本アーカイブを可能な限りそのまま保存する。
* 原作者の著作権表示・クレジットを尊重する。
* 原本ソースは変更しない。
* UTF-8化、JavaScript移植、翻訳、現代向け改良版については別プロジェクトとして扱う。

## 著作権について

原本には明示的なオープンソースライセンスは確認できませんでした。

著作権は原作者に帰属します。

`miraiscript/jcodeg_pl.txt` について: 本ファイルは、歌代和正さんに
よるオリジナルの`jcode.pl`、gamaさんの`getcode.pl`が持つ
`getcode()`ルーチン、および「町の妖精さん」による改変が重なった
派生物です。改変者自身のコメントにより、改変部分について追加の
編集著作権は主張しない旨が明記されていますが、原著作物の著作権は
元の著作者に帰属します。

本リポジトリは、歴史的・研究的資料としてアーカイブを保存するものであり、原著作物の権利を主張するものではありません。

権利者から修正・削除の申し出があった場合は、誠実に対応いたします。
